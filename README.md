## Pandas Examples

This repository contains example code based on the NumPy tutorial by Bro Code.  
All examples are separated into multiple files for better understanding and practice.

  Source: https://www.youtube.com/watch?v=VXtjG_GzO7Q

---

## Files

- `introduction.py` → Just start import
- `series.py` → Working with Series
- `dataframe.py` → Working with DataFrame (2D table)
- `importcsv.py` → Reading CSV / loading data
- `selection.py` → Column & row selection (loc, iloc)
- `filtering.py` → Conditional filtering (Boolean indexing)
- `aggregatefunction.py` → Sum, mean, min, max (Aggregation)
- `datacleaning.py` → Handling missing data / cleaning

---

## 🚀 Getting Started

### 1. Create Virtual Environment (optional)
```bash
python -m venv .venv
source .venv/Scripts/activate
```

### 2. Install pandas
```bash
pip install pandas
```
